create-react-app을 이용한 최초 설치시 

```
//https://github.com/facebook/create-react-app

npx create-react-app movie-app
cd movie-app
```

이미 movie-app diretory가 존재 하는 경우

```
npm install
yarn start //npm start
```

deploy

```
yarn add gh-pages
yarn run deploy
```
